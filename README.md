## Build Setup

``` bash
# 安装依赖
npm install

# 启动本地测试服务，访问地址：localhost:8080
npm run dev

# 打包生产文件
npm run build
```

## Project Structure

* `src/assets` - 静态资源文件夹
* `src/components` - 自定义vue组件文件夹
* `src/css` - 通用css样式文件夹
* `src/pages` - 核心项目文件夹
* `src/main.js` - 入口js文件
* `src/routes.js` - 项目路由文件
* `src/app.vue` - 入口vue文件
